# Descripción
Sistema para la gestión de una clínica veterinaria que permite administrar acudientes, mascotas e historiales clínicos, facilitando el registro y consulta de la información.

# Tecnologías
 - Java 21
 - Spring Boot
 - Spring Data JPA
 - Maven

# Ejecución
Clonar repositorio:
    git clone https://github.com/tu-usuario/clinica-veterinaria.git

Crear base de datos:
    Ingresar a pgAdmin y crear la base de datos clinica

Ejecutar:
mvn spring-boot:run
